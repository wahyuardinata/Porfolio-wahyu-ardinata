My Skills

HTML

Saya memulai dengan HTML sebagai dasar dalam membangun struktur halaman web. Saya belajar tentang elemen-elemen penting seperti heading, paragraph, table, form, dan lainnya. HTML menjadi pondasi utama dalam pengembangan web yang saya pelajari lebih lanjut dengan teknologi lain.

CSS

Setelah memahami struktur HTML, saya belajar CSS untuk mendesain tampilan web agar lebih menarik. Saya mempelajari berbagai properti seperti warna, layout, positioning, flexbox, dan grid. Selain itu, saya juga memahami konsep responsivitas agar tampilan web tetap nyaman di berbagai perangkat.

JavaScript

Saya mulai belajar JavaScript untuk menambahkan interaktivitas pada halaman web. Saya memahami konsep dasar seperti variabel, fungsi, event, DOM manipulation. Dengan JavaScript, saya dapat membuat elemen dinamis seperti animasi, validasi form, dan fitur interaktif lainnya.

Bootstrap

Untuk mempercepat pengembangan antarmuka, saya menggunakan Bootstrap sebagai framework CSS. Saya belajar menggunakan komponen seperti navbar, card, modal, dan grid system untuk membuat desain yang lebih modern dan responsif tanpa harus menulis banyak kode CSS manual.

PHP

Saya mempelajari PHP untuk mengembangkan sisi backend dari aplikasi web. Saya belajar tentang sintaks dasar, penggunaan variabel, fungsi, session, dan cookie. Selain itu, saya juga memahami cara menghubungkan PHP dengan database untuk menyimpan dan mengambil data dari server.

Data Base

Saya mempelajari konsep database menggunakan MySQL. Saya memahami cara membuat tabel, relasi antar tabel, serta melakukan operasi CRUD (Create, Read, Update, Delete) dengan SQL. Saya juga belajar tentang optimasi query dan penggunaan indeks untuk meningkatkan performa database.
